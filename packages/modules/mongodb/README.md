# MongoDB Module

This module provides MongoDB integration for the Stray Rugs application using Mongoose.

## Features

- User model definition
- Database connection utilities

## Technologies Used

- MongoDB
- Mongoose
- TypeScript

## Models

### User

- firstName (string)
- lastName (string)
- email (string)
- password (string, hashed)
- isActive (boolean)
- role (string: "user" | "admin")
